# projeto-calendario
## Descrição
Sistema de gestão de calendário pessoal desenvolvido em Python, usando POO e persistência em JSON.

## Funcionalidades
- Adicionar eventos
- Validar datas
- Detetar sobreposição
- Persistência em ficheiro JSON
